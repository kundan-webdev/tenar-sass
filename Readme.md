# Tenar — Compliance Monitoring SaaS Landing Page

A production-ready landing page for **Tenar**, a compliance monitoring and alert system built for Indian exporters. Designed with a premium dark aesthetic and smooth user experience.

> Coming soon

---

## ✨ Features

- **Dark Forest UI** — custom design system with CSS variables (`--forest`, `--copper`, `--sage` color palette)
- **Early Access Modal** — animated modal with form validation, success state, and smooth transitions
- **Fully Responsive** — mobile-first layout with breakpoints at 480px, 600px, and 1000px
- **Custom Scrollbar** — thin copper-tinted scrollbar matching the brand aesthetic
- **Scroll Animations** — intersection observer-based reveal animations on all sections
- **Interactive Hero Panel** — live stats dashboard mockup with animated progress bars
- **Testimonial Slider** — auto-rotating quote carousel with dot navigation
- **Pricing Section** — 3-tier pricing cards with highlighted recommended plan
- **Custom Cursor** — mycelium canvas cursor with hover and click states
- **Ticker Strip** — infinite-scroll compliance regulation marquee
- **All Indian States Dropdown** — all 28 states + 8 Union Territories, alphabetically sorted

## 📱 Responsive Breakpoints

| Breakpoint | Layout Change |
|---|---|
| `< 1000px` | Single column hero, hidden right panel |
| `< 600px` | Single column features, pricing, impact grids |
| `< 500px` | Single column form rows, reduced modal padding |
| `< 480px` | Bottom-sheet modal, full-width layout |
